# Controlador de temperatura para calderas
El proyecto que se desarrolla a continuación consiste en un dispositivo que se utiliza en calderas para el control de la temperatura. Más específicamente, permite realizar el control para calderas de tipo doble servicio, donde se maneja tanto agua sanitaria como calefacción por losa radiante. La importancia de este dispositivo es que permite reemplazar los controladores utilizados actualmente que son de origen extranjero, ofreciendo una alternativa local que ahorre los procesos de importación. Además, debido a que es de código abierto, es posible realizar ajustes según las necesidades de cada consumidor.
